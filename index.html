Cómo se arma el rating. Cada país tiene un rating combinado = 82% Elo (eloratings.net) + 18% señal de mercado. La señal de mercado mezcla el % de campeón del modelo de Goldman Sachs (research del 29-may-2026, fijo) con el % de campeón de Polymarket, que se trae en vivo desde su API pública cada vez que abrís la página o tocás "Actualizar". Ese rating alimenta un motor Elo-Poisson que calcula goles esperados, marcador más probable, 1·X·2 y quién avanza.
Sobre Betano. Las casas de apuestas tradicionales no tienen API pública, así que no se puede conectar en vivo — por eso no aparece como fuente separada acá; el consenso de mercado se apoya en Polymarket, que sí es público.
Ojo. Es una herramienta de estimación, no consejo de apuestas. El fútbol es ruido.
